# Digital Clock

A responsive digital clock built with HTML, CSS, and JavaScript. The application displays the current time, live date, and a greeting message based on the time of day. The clock updates automatically every second using JavaScript's Date object and setInterval().

## Features

- Displays the current time in HH:MM:SS format
- Updates automatically every second
- Shows the current day, date, month, and year
- Displays a dynamic greeting:
  - 🌄 Good Morning
  - ☀️ Good Afternoon
  - 🌙 Good Evening
- Responsive and clean user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Date Object
- DOM Manipulation
- setInterval()
